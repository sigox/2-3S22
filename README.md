# 開始使用

1.請先確認有安裝 node.js 與 npm

2.將專案 clone 到本地

3.在本地開啟之後，透過終端機進入資料夾，輸入：

    node app.js

若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

Listening on http://localhost:3000

若欲暫停使用

ctrl + c

# 開發工具

- Node.js 10.15.0
- Express 4.17.1
- Express-Handlebars 5.3.3
- Bootstrap 5.2.1
